Comment about project

The website was developed using AngularJs + Bootstrap + CSS + JavaScript + JSON. The layout (design) is custom (the idea is based on one of the free Bootstrap templates).

Initially, the website was considered as an educational project on which they worked out: programming, design and working with the framework, then (virtually unchanged) it was implemented as a commercial project for the local funeral services bureau, where it functioned as a business card website for four years.

Коментарі щодо проекту

Сайт розроблено з використанням AngularJs + Bootstrap + CSS + JavaScript + JSON. Верстка власна (за основу брався один з безкоштовних шаблонів Bootstrap).

Спочатку сайт розглядався, як навчальний проект де віпрацьовувались: програмування, верстка та робота з фреймворком, після чого (практично без змін) був реалізований як комерційний проект для місцевого бюро ритуальних послуг, де в якості сайта-візитки функціонував на протязі чотирьох років.

Коментарии по проекту

Сайт разработан c использованием AngularJs + Bootstrap + CSS + JavaScript + JSON. Верстка кастомная (в основе идеи лежит один из бесплатных шаблонов Bootstrap).

Первоначально сайт рассматривался, как учебный проект на котором отрабатывались: программирование, верстка и работа с фреймворком, затем (практически без изменений) был реализован как коммерческий проект для местного бюро ритуальных услуг, где в качестве сайта-визитки функционировал в течение четырех лет.
